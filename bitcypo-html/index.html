How To Buy

1. DOWNLOAD METAMASK
For this tutorial, we recommend using the MetaMask wallet, which can be installed as a Chrome extension or downloaded as a mobile application.
2. CONFIGURE FOR THE BSC NETWORK
Next up we configure MetaMask to work with BSC, meaning we can trade Binance Coin (BNB) for BTC minion coin using MetaMask. Heres a handy how to guide: Add BSC To MetaMask
3. BUY SOME BNB
‍Now you have the wallet set up, you can add some BNB and begin trading. Head to any exchange you are registered to(Coinbase, kucoin, binance, kraken) and purchase BNB using your native currency
4. SEND YOUR BNB TO METAMASK
‍Open up your MetaMask wallet and copy your wallet address – you can find this underneath your account name, it will start with "0x". Head to your exchange and withdraw your BNB to your MetaMask wallet address. Send on the BEP20 network.
5. TRADE YOUR BNB FOR CROP
Go to poocoin using the buy now button below. btc minion is already selected by default. Connect your wallet by using the “Connect” button in the top right hand corner. Then push trade. Ensure the "Auto slippage" button is green and then input the number of BNB you would like to trade and hit swap.
